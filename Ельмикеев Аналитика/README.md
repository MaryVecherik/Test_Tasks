# Тестовые задания — Junior Analyst Power BI

### 📌 Описание
Данный репозиторий содержит решение тестовых заданий на позицию **Junior Analyst Power BI**.

Проект состоит из двух этапов:
- **Этап 1** — базовая аналитика продаж
- **Этап 2** — бизнес-дашборд для Wildberries (финансы)

Проект демонстрирует:
- навыки работы с Power BI
- построение модели данных
- написание DAX-мер
- визуализацию бизнес-метрик
- работу с БД и Google Sheets
